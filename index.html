Vertical Axis Wind Turbine (VAWT) Design

Senior Design Project

• Designed a compact Vertical Axis Wind Turbine (VAWT) aimed at enhancing renewable energy generation in urban environments with limited space. The project focused on creating an efficient and space-saving turbine that could perform well in varying wind conditions commonly found in cities. Using Fusion360, I developed detailed 3D models and conducted simulations to optimize the turbine's aerodynamic performance. I created physical prototypes using 3D printing for structural components and fiberglass for the blades. MATLAB was employed for data analysis and evaluating the turbine’s performance. I led the entire design process, from concept development and prototyping to testing. The final prototype was successfully built and tested, showing promising results under different wind conditions. Throughout the project, I gained valuable experience in aerodynamic design, manufacturing techniques, and field testing, which deepened my understanding of both engineering principles and practical implementation challenges.

Servo Motor Control System

Course Project

• This system is a multi-input control setup that allows a servo motor to be operated both manually and automatically. The user can directly control the servo angle using a potentiometer in manual mode, while in automatic mode, the system uses a photoresistor (LDR) to adjust the servo’s movement speed based on the ambient light level. A push button is used to toggle between these two modes. The project is designed to teach fundamental concepts such as sensor reading, mode switching, and servo control in Arduino-based systems. It is suitable for educational purposes and can also be applied in light-sensitive automation, robotics, and prototype development projects, making it both a practical and instructive system.
